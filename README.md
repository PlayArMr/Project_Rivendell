# Project Rivendell

A terminal-based text adventure game written in C. Explore interactive environments and make choices that shape your story.

## Features

- **Text-Based Adventure**: Immersive narrative gameplay in your terminal
- **Interactive Choices**: Your decisions matter
- **Multiple Locations**: Explore diverse environments
- **Terminal-Only**: Play anywhere with just a console

## Getting Started

### Run the Game

Pre-compiled executables are included:

```bash
./game
```

### Build from Source

```bash
gcc main.c forest.c village.c credits.c -o game
./game
```

## Project Structure

- `main.c` - Game entry point
- `forest.c` - Location module
- `village.c` - Location module
- `credits.c` - Credits
- `game` - Compiled executable
